Deep Dive: Preprocessing

Standardizing inputs for model consistency.

  • Auto-Contrast: ImageOps.autocontrast(pil_img) maximizes dynamic range.
  • Denoising: cv2.fastNlMeansDenoisingColored (h=3, hColor=3) removes grain.
  • CLAHE: Contrast Limited Adaptive Histogram Equalization (Clip=2.0, Tile=8x8) enhances local details.
  • Grayscale Conversion: Reduces channel noise before re-stacking to RGB.

MBConv Blocks

The core building block of EfficientNet.

  • Inverted Residuals: Expands channels -> Depthwise Conv -> Projects back.
  • Depthwise Separable Conv: Reduces parameters by ~9x compared to standard conv.
  • Skip Connections: Preserves gradients across deep networks.

Lung Disease Model

Custom Keras/TensorFlow Sequential CNN.

{ "class_name": "Sequential", "config": { "layers": [ ... ] } }

Designed specifically for 3-class classification: Normal, Benign, Malignant.

Feature Extraction

  • Block 1: Conv2D (128 filters, 8x8 kernel) -> BatchNorm -> ReLU
  • Block 2: Conv2D (256 filters, 5x5 kernel) -> BatchNorm -> ReLU -> MaxPool
  • Block 3: Conv2D (256 filters, 3x3 kernel) -> BatchNorm -> ReLU
  • Block 4: Conv2D (512 filters, 3x3 kernel) -> BatchNorm -> ReLU -> MaxPool

Regularization Strategy

Preventing overfitting in medical data.

Batch Normalization

Applied after every Conv layer to stabilize learning.

Dropout (0.5)

Applied in dense layers to force redundant feature learning.

Dense Classifier

Flatten() Dense(1024, activation='relu') Dropout(0.5) Dense(1024, activation='relu') Dropout(0.5) Dense(3, activation='softmax')

High-capacity fully connected layers to map features to disease classes.
